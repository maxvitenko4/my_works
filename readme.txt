Project description

This project analyzes sales data from an e-commerce dataset.
The goal is to build a dashboard that shows overall sales performance, trends over time, and key business metrics.
The dashboard was created as a portfolio project to demonstrate basic skills in SQL, BigQuery, and Tableau.

Data source
Dataset: thelook_ecommerce (public dataset)
Platform: Google BigQuery
Tables used:
orders;
order_items;
products;
users

Tools
SQL (Google BigQuery)
Tableau Public

KPI
The following metrics are calculated in the dashboard:
Revenue – total sales amount
Orders – number of unique orders
AOV – average order value
Customers – number of unique customers

Dashboard content
The dashboard includes:
Monthly sales trend
Sales by country (map)
Sales by product category
Summary table with revenue, orders, customers, and average order value
Filters by year, country, and category

Business questions
The dashboard helps answer the following questions:
How do sales change over time?
Which countries generate the most revenue?
Which product categories perform best?
